Where Do I Sort and Filter Items?

In the left menu of To Do Machine, there are multiple sections of buttons. The bottom section, labeled, "Sorting and Filtering a List", contains four buttons: "Sort by Date", "Show All Items", "Show Completed Items", and "Show Incomplete Items". Clicking on either of these buttons will either sort the list by date or will show only certain items, filtering them.

Sorting

By clicking on the "Sort by Date" button, the table will be sorted in a descending order of date, whereas the item with the earliest date provided will be at the top of the list. For items without a date, they are placed beneath all items with a date, as there are no times for those items to be completed by.

Filtering

The list can be filtered by its complete status by using the three buttons beneath the "Sort by Date" button.

In filtering the list, items that are not shown are not deleted, but rather are hidden from view. Exporting a list with filtered elements will still export them as expected.

Show All Items

The button labeled "Show All Items" causes all of the items in the table to appear, meaning any previously filtered items will appear in their previous positions as expected.

Show Completed Items

The button labeled "Show Completed Items" causes all of the incomplete items in the table to be hidden, leaving only the items marked as complete to be modifiable and seen.

Show Completed Items

The button labeled "Show Incomplete Items" causes all of the complete items in the table to be hidden, leaving only the items marked as incomplete to be modifiable and seen.
