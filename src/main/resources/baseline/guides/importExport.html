Where Do I Import and Export?

In the left menu of To Do Machine, there are multiple sections of buttons. The top section, labeled, "Import and Export a List" contains two buttons: "Import To-Do List" and "Export Current To-Do List". Clicking on one of these buttons will either import a list or export a list respectively.

In addition, the context menu at the top of the application contains a menu item named "File", which can be clicked for access to both import and export functionality.

Importing a List

Clicking the "Import To-Do List" button or the import function from the context menu will open up a dialog box prompting the user to select a text file from their storage directory. After selecting a file, the program will import the data stored within the file to the table, and a popup will appear after the data is imported. The imported data may then be modified at the user's will.

WARNING: IF THE TEXT FILE PROVIDED IS NOT IN THE CORRECT FORMAT, THE PROGRAM WILL NOT IMPORT PROPERLY, AND AN ERROR WILL APPEAR MENTIONING THAT THE FILE COULD NOT BE IMPORTED.

Due Date

Clicking the "Expokrt Current To-Do List" button or the export function from the context menu will open up a dialog box prompting the user to enter a file name and a destination for the to the to-do list to be saved to. After the user presses "Save", the program will export a text file with the list's contents to the destination specified, and a popup will appear mentioning the successful export.
